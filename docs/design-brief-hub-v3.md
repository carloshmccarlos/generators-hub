# Design Brief: Homepage Redesign - Editorial Magazine Style

## 1. Feature Summary
Generator Hub 主页面重构，Editorial 杂志风格。面向内容创作者、社交媒体管理者、数据分析师和开发者。核心功能：展示并导航到可用的生成器工具。

## 2. Primary User Action
快速发现并访问生成器工具。

## 3. Design Direction
**Editorial Magazine** - 杂志风格

- **情绪**：高端、精致、透气
- **参考文献**：Vogue, Kinfolk, Apple Pro
- **视觉**：大胆排版 + 大量留白 + 暖奶油色背景

## 4. Layout Strategy
**Hero Section**：
- 大标题（居中或左对齐）
- 简短描述

**工具卡片网格**：
- 2列网格桌面 / 1列移动
- 每卡片：状态标签 + 名称 + 描述 + CTA

**Coming Soon**：
- 视觉区分，简洁设计

## 5. Key States
- Default: 2 live tools
- Hover: 边框高亮 + 轻微位移

## 6. Implementation
- 字体：Bricolage Grotesque + Manrope（避开禁止列表）
- 颜色：暖奶油色背景 + 品牌强调色
- 动画：交错淡入
